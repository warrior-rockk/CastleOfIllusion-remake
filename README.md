# Castle of Illusion (remake)
Remake del juego Castle of Illusion de Sega Master System desarrollado en BennuGD [https://www.bennugd.org](https://www.bennugd.org), un fork del conocido Div Games Studio.

La idea es hacer un engine de plataformas 8bit para BennuGD y para ello se implementa un remake del juego de Disney para la Master System.

Contiene un nivel completo y un nivel nuevo que no existía en el juego original.
